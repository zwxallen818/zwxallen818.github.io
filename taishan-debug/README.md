## 本版（给他人使用）

- **默认须连接蓝牙** 后才能进遥控（加载页可手动开「跳过蓝牙」仅供自测）
- 遥控页返回：断开蓝牙 + 回 Global App 设备列表
- 未连蓝牙时摇杆不发 `move`，减少 Bridge 报错
- 已完成本月引导的用户不再绕过 CheckRouter 直进遥控

若仍免连蓝牙进入：在加载页关闭「跳过蓝牙」，或 **删除 App 内旧 w3 包** 后重装本版。
