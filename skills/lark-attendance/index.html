飞书考勤打卡查询

lark-attendance

查询飞书考勤打卡记录，支持按日期筛选。依赖 lark-cli 和 lark-shared 认证。

版本 1.0.0更新 2026-05-25来自 Cursor

何时启用

  • 查询自己的考勤打卡记录
  • 查看某天是否打卡、迟到、早退
  • 统计加班时长

API 调用

# 查看 API 参数结构
lark-cli schema attendance.user_tasks.query

# 查询考勤记录
lark-cli attendance user_tasks query \
  --params '{"employee_type":"employee_no"}' \
  --data '{"user_ids":[], ...}'

默认参数

  • employee_type 固定为 "employee_no"
  • user_ids 固定为空数组 []
  • 以上参数禁止向用户询问

安装（全局）

git clone git@github.com:zwxallen818/cursor-skills.git
cd cursor-skills
cp -r skills/lark-attendance ~/.cursor/skills/

前置依赖

需要安装 lark-cli 并配置认证，详见 lark-shared Skill。

# 需要先安装以下依赖 Skill
cp -r skills/lark-shared ~/.cursor/skills/
cp -r skills/lark-attendance ~/.cursor/skills/

# 需要 lark-cli 命令行工具
# 权限 scope: attendance:task:readonly

注意事项

  • 使用前先读取 lark-shared 认证配置
  • 调用 API 前必须先 schema 查看参数
  • 不要猜测字段格式
