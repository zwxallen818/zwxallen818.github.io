乱稿 Figma → 响应式实现

messy-figma-responsive

从混乱 Figma 稿实现响应式 UI。截图优先、可见即所得、flex 布局与 margin 内容沟。

版本 1.0.0更新 2026-05-23来自 Cursor

何时启用

  • 用户提供含 node-id 的 Figma 帧链接
  • 设计稿乱、绝对定位、切图无规则
  • 不要把 get_design_context 整页输出当最终代码

执行流程

Phase 0 — 侦察截图 → ref-full.png，写 AUDIT.md
Phase 1 — 划区语义区块，写 BLOCKS.md
Phase 2 — 切图判定slice-manifest.ts
Phase 3 — 骨架代码flex 布局，禁止整页 absolute
Phase 5.5 — 同宽对比414px Playwright 对比

安装（全局）

git clone git@github.com:zwxallen818/cursor-skills.git
cd cursor-skills
cp -r skills/messy-figma-responsive ~/.cursor/skills/

硬规则

  • 仅实现 screenshot 可见内容
  • 不实现 Status Bar、Home Indicator
  • 禁止 max-w-[390px] 锁死画布
