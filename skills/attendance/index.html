考勤加班时间查询

attendance

从 Gaia Workforce 系统获取考勤数据，自动计算加班时长。输出 RDM 工时填报汇总。

版本 1.0.0更新 2026-05-25来自 Claude Code

何时启用

  • 查询上周加班时间，用于 RDM 工时填报
  • 查看某天是否打卡、迟到、早退
  • 统计本月/上周工时汇总

核心命令

# 查询上周加班（RDM 工时填报）
python3 ~/.claude/skills/attendance/fetch_attendance.py last_week

# 查询本周考勤
python3 ~/.claude/skills/attendance/fetch_attendance.py this_week

# 查询本月考勤
python3 ~/.claude/skills/attendance/fetch_attendance.py this_month

# 自定义日期范围
python3 ~/.claude/skills/attendance/fetch_attendance.py \
  --start 2026-05-01 --end 2026-05-15

# 强制刷新 cookies
python3 ~/.claude/skills/attendance/fetch_attendance.py --refresh

# 首次使用：设置登录凭据
python3 ~/.claude/skills/attendance/fetch_attendance.py --setup-credentials

安装

git clone git@github.com:zwxallen818/cursor-skills.git
cd cursor-skills
cp -r skills/attendance ~/.claude/skills/

依赖安装

pip install requests playwright requests-ntlm
playwright install chromium

认证方式

NTLM SSO + Playwright。自动完成 Windows NTLM 认证 → SSO 跳转 → Keycloak 会话 cookie。cookies 过期时自动刷新。

加班计算规则

  • 工作日：18:30 之后算加班，向下取整到 30 分钟
  • 休息日：下班时间 - 上班时间 = 全部算加班
  • 请假识别：自动检测请假/休假/年假等关键词
