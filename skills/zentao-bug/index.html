禅道 Bug 管理

zentao-bug

通过 CLI 管理禅道 Bug：查询、解决、指派。支持 myecovacs 定制枚举，解决时默认指派回提单人。

版本 1.0.0更新 2026-05-25来自 Cursor

何时启用

  • 查看分配给自己的禅道 Bug
  • 解决 / 指派 / 设置解决方案
  • zentao.myecovacs.com 站点专用

核心命令

# 检查认证
python scripts/zentao_cli.py auth check

# 查看我的 Bug
python scripts/zentao_cli.py bugs mine --limit 100

# 查看 Bug 详情
python scripts/zentao_cli.py bugs get 12345

# 解决 Bug（默认指派回提单人）
python scripts/zentao_cli.py bugs resolve 12345 \
  --resolution solved \
  --assigned-to OPENER_ACCOUNT \
  --comment "已解决，请提单人核验"

# 指派给其他人
python scripts/zentao_cli.py bugs assign 12345 \
  --to other_user --comment "转交处理"

解决方案枚举（myecovacs）

  • solved — 已解决（不要用 fixed）
  • no — 不解决
  • notrepro — 无法重现
  • invalid — 无效缺陷
  • postponed — 推迟解决

安装（全局）

git clone git@github.com:zwxallen818/cursor-skills.git
cd cursor-skills
cp -r skills/zentao-bug ~/.cursor/skills/

环境变量

在 skill 目录下创建 .env 文件（勿提交仓库）。

ZENTAO_BASE_URL="https://zentao.example.com/"
ZENTAO_USERNAME="your_account"
ZENTAO_PASSWORD="your_password"

安全工作流

  • 修改前先 bugs get 查看详情
  • 解决时默认指派回提单人
  • 确认后再加 --yes
  • 密码不打印、不持久化
